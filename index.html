You are converting raw novel text into a line-by-line audiobook script. Rules: - Every line must start with "CharacterName: " - For characters whose name is already known from the START of the chapter, use their name (Subaru, Puck, Narrator, etc.) - For unnamed or mystery characters, assign them a number: Character1, Character2 etc. — keep the number consistent throughout the entire chapter - When a mystery character's real name is REVEALED during the chapter, switch to using RealName[N] format for ALL their remaining lines — e.g. Emilia[1], Guard[2]. This links them to their earlier CharacterN lines so the same voice is used - IMPORTANT: If a character is introduced by name at the start (e.g. Emilia), they are just "Emilia" — do NOT give them a number. Only characters who start the chapter unnamed get numbers - IMPORTANT: If two different character names refer to the same person (e.g. a character is called "Satella" in one chapter but their real name "Emilia" is revealed later), treat them as the same character and use the number format consistently - All narration, descriptions, and action lines use "Narrator:" - Internal thoughts use the thinking character's name - Skip translator notes like [T/N: ...] - Output ONLY the formatted lines, no explanations, no blank lines between - This text will be read aloud by an AI voice. To create natural pauses where a human speaker would breathe or pause for effect, insert extra commas (,,) at key moments — such as after an em dash (―), before a dramatic reveal, after an ellipsis, or where a speaker would naturally pause for emphasis. Do not add commas randomly — only where a real pause would feel natural when spoken aloud Paste your chapter text below this line, then hit send: ---
